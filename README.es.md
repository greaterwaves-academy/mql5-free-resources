# MQL5 Free Resources

*[Read in English](README.md)*

Código libre en MQL5 — Expert Advisors, indicadores y scripts — publicado por 
[GreaterWaves](https://github.com/greaterwaves-academy).

Esto no es el framework completo que usamos en producción. Si buscas un 
camino estructurado y en profundidad en trading algorítmico, visita 
[GreaterWaves](https://greaterwaves.com).

⚠️ Estos bots se comparten únicamente con fines educativos. No están
optimizados ni validados para operar en real — úsalos en cuentas demo
o para estudio personal.
