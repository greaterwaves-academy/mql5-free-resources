# MQL5 Free Resources

*[Leer en español](README.es.md)*

Free MQL5 code — Expert Advisors, indicators, and scripts — published by 
[GreaterWaves](https://github.com/greaterwaves-academy).

This is not the full framework used in production. For a structured, 
in-depth path into algorithmic trading, check out 
[GreaterWaves](https://greaterwaves.com).

⚠️ These bots are shared for educational purposes only. They are not
optimized or validated for live trading — use them on demo accounts or
for study.
